# Live streaming platform
## Installation

- Install packages <br/>
  ```npm install```

## Setup

- Run the backend app in development mode <br/>
  ```npm run dev:backend```
- Run the frontend app in development mode <br/>
  ```npm run dev:web```
- Run the RTMP Server & HLS transcoder [important]: <br/>
  1. ```docker-compose build``` <br/>
  2. ```docker-compose up```
